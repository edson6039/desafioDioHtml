  1. Conceitos sobre clientes

    2. Client é quando você consome a informação de algum lugar, basicamente é o dispositivo que está consumindo a informação da internet.

    O termo hospedar significa que você vai colocar o arquivo no servidor.

    O cache é quando os arquivos estáticos são armazenados no seu computador, pelo navegador para que não precise baixar novamente, deixando assim mais veloz o carregamento do site. Diferentes resoluções, deve ser considerado na criação de websites, design responsivo ou responsividade, também a compatibilidade entre navegadores (Browsers), devem ser feitas boas práticas e testes.

    Navegadores são programas criados por empresas, utilizados para abrir/executar arquivos. Também são chamados de browsers. Eles seguem padrões W3C, porém, sempre tem uma diferença ou outra de interpretação. São sempre gratuitos.

    Aplicações web são soluções criadas que possuem a internet como meio de comunicação entre cliente x server não sendo necessária a instalação.

  2. Conceitos sobre servidores

    3. O servidor é a máquina central, é o computador que serve as informações necessárias para os outros computadores. Em uma empresa quando existe um computador central que cuida das informações dos outros, ele é um servidor central. Os servidores tendem a ser mais robustos pois precisam processar mais informações.

    Existem servidores de arquivos, segurança, streaming, e-mail, web. Data centers são lugares onde são armazenados vários servidores em conjunto, para servir a diversos clientes ao mesmo tempo.

    Diferenças entre hardwares e softwares:

    Softwares: Sistemas operacionais, monitoramento, servidores web. Hardwares: Armazenamento (discos rígidos, ssds), memória, processadores.

    Como funcionam os servidores: Ocorre uma requisição HTTP do cliente para o servidor, quando essa requisição chega, o sistema operacional executa, processa e devolve a devida resposta para o cliente.

  3. Tipos de servidores

    • Proxy: Ele é um servidor que serve o acesso à internet para outros computadores porém bloqueia o acesso de alguns sites. Ele também pode ser um servidor de autenticação, usuário e senha.
    • Firewall: Ele também serve para essa segurança, mas ele é mais voltado para o acesso externo tentando acessar a rede interna.
    • Web Server: É um servidor de acesso a internet. Exemplo: Apache Tom Cat.
    • Email Server: Um servidor voltado para armazenamento de e-mails.
    • DataBase Server: Um servidor de banco de dados é aquele que armazena as informações de pessoas, produtos e etc...
    • Intranet: É um Servidor Web sem internet, ele só é possível acessar na rede local de computadores.
    • DNS (Domain Name Service): Ele é um servidor que possui uma lista de nomes e IPs, esse servidor define o IP de cada um dos computadores conectados.

    Hospedagem de sites: É colocar os seus arquivos de um site em um servidor que fique disponível na internet para seus clientes utilizarem.

  4. O que é linguagem de programação

    5. Linguagem de programação é uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (software).

    Linguagem de programação Server-side é uma linguagem executada no lado do servidor. Exemplo: Php, Java, Javascript, Ruby, C#.

    Linguagem de programação Client-side é uma linguagem executada no lado do cliente. Exemplo: Chrome, node, javascript.
